# CodeCompose

**Format & Share Code in Gmail.**

CodeCompose is a Chrome extension that lets you highlight any code in a Gmail compose window, right-click, and instantly insert a beautifully formatted, syntax-highlighted code block — without leaving Gmail.

---

## Features

- **Syntax highlighting** — 19 languages including JavaScript, Python, Java, PHP, Go, Rust, and more
- **Code formatting** — js-beautify (Free) and Prettier (Junior Dev+)
- **5 themes** — Dark, Light, Dracula, GitHub, Monokai
- **Auto language detection** — intelligently identifies the language from your code
- **Line numbers** — clean, fixed-width column (Senior Dev)
- **Copy to clipboard** — copy formatted code directly from the insert dialog (Senior Dev)
- **Email-safe output** — fully inline styles, renders correctly in Gmail, Outlook, and Apple Mail
